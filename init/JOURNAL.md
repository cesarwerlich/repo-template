# Journal

Chronological notes for work that should be written down but does not yet belong in `MEMORY.md`, an ADR, or a runbook.

## How To Use

- Record dated notes about direction changes, discoveries, and important discussion outcomes.
- Move stable truths and repeated decisions into `MEMORY.md`.
- Link to deeper docs when a note grows beyond a quick entry.

## Entries

```text
YYYY-MM-DD - Topic
- Context:
- What changed:
- Decision:
- Follow-up:
```
