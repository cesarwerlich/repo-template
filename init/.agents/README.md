# Agent Subsystem

Optional guidance for AI coding agents.

## How To Use

1. Read the project root `CONTEXT.md`, `ROADMAP.md`, and `MEMORY.md`.
2. Pick a relevant skill from `skills/` only after understanding the task.
3. Use personas in `agents/` for focused review perspectives.
4. Use `references/` for checklists when a skill asks for deeper support.

Skills and personas are helpers. Local project instructions in `AGENTS.md` always take precedence.
