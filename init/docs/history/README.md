# History

Archive for older `journal.md` entries and retired logs. Keeps the active
journal small so session startup stays cheap.

## How To Use

- When `journal.md` grows long, move stale entries here, grouped by period
  (e.g. `2025-q1.md`).
- This directory is read rarely and on purpose — never part of session startup.
- Prefer `git log` for fine-grained execution history; use this only for curated
  narrative worth keeping at a glance.
